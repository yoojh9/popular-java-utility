<b> <a href="Md5Hex.java"> 1. md5Hex() </a> </b>

 - MD5 digest를 계산하고 16 바이트 byte[]로 반환
<br>

<b> <a href="Sha256Hex.java"> 3. sha256Hex() </a> </b>
 - SHA-256 digest를 계산하고 값을  16진수 문자열로 반환
<br>
